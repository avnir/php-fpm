# php-fpm docker image

This is a [Docker](http://www.docker.com) image using [PHP-FPM](http://php-fpm.org/).

`docker pull avnir/php-fpm:php5`
`docker pull avnir/php-fpm:php7.0`
`docker pull avnir/php-fpm:php7.1`
`docker pull avnir/php-fpm:php7.2`
